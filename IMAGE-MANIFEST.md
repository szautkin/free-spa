# üì∏ Complete Image Manifest

## Overview

This document lists **ALL images** needed for the Small Business SPA Template. Simply drop your images into the appropriate folders following these naming conventions.

**Base Directory:** `public/assets/[business-type]/`

Each business type has its own folder:
- `coffee` - Coffee Shop
- `restaurant` - Restaurant
- `bakery` - Bakery
- `boutique` - Boutique/Retail
- `salon` - Salon/Spa

---

## üéØ Quick Start

1. Choose your business type
2. Navigate to `public/assets/[your-business-type]/`
3. Follow the naming conventions below
4. Drop your images into the correct folders
5. Your site automatically uses them!

## üé® Image Guidelines

### Recommended Sizes
- **Hero Background:** 1920√ó1080 (Full HD) - Wide, horizontal
- **Logo Files:** 512√ó512 (square) or transparent background
- **Category Images:** 800√ó600 (4:3 ratio) - Landscape
- **Product Items:** 400√ó400 (square) - Consistent grid
- **Gallery Images:** 1200√ó800 (3:2 ratio) - Landscape
- **Gallery Thumbnails:** 300√ó200 (3:2 ratio) - Smaller versions
- **Favicon:** 32√ó32 (tiny square)

### Format Guidelines
- **Photos:** JPG (compressed, smaller file size)
- **Logos:** PNG (transparency support)
- **Icons:** SVG (scalable, crisp at any size)

### File Size Optimization
Compress all images before adding them:
- Hero images: < 500KB
- Gallery images: < 300KB
- Thumbnails: < 50KB
- Product images: < 150KB

**Free compression tools:**
- TinyPNG
- Squoosh
- ImageOptim (Mac only)

## ‚ùì FAQ

**Q: What if I don't have all these images?**
A: Start with the required images (hero, logo, about-main, 5 menu categories, 12 gallery images). The template will use the placeholder image for any missing files.

**Q: Can I use different names?**
