A complete, professional website with:
- Beautiful homepage with your photos

- Mobile-friendly design
- Easy to customize

## Quick Links

**Files You'll Edit:**
- `src/config/content.config.ts` - All your text and menu items
- `src/config/theme.config.ts` - Colors, fonts, and design
- `src/config/assets.config.ts` - Links to your images
- `public/assets/coffee/images/` - Your actual photos

**Most Common Tasks:**
- Change business name → SETUP-GUIDE.md (Step 2)

## Technical Requirements

**To preview your site locally:**
- Node.js installed (version 16 or higher)
- A code editor (VS Code recommended, free download)
- A modern web browser (Chrome, Firefox, Safari, Edge)

**Commands to know:**
```bash
npm install          # First time only - installs dependencies
npm run dev         # Preview your site locally
```

Your site will be available at: `http://localhost:5173`

│   │   ├── content.config.ts    ← Your text content

│   └── assets/
│       └── coffee/

**Simple rule:** Only edit files in `src/config/` and add images to `public/assets/coffee/images/`. Leave everything else alone!

## Common Questions

### "Do I need to know code?"
No! You'll be editing simple configuration files with clear instructions.

### "What if I break something?"
You can always undo (Ctrl+Z / Cmd+Z) or restore from the original template.

### "How long does this take?"
- Basic setup: 30-60 minutes
- Adding all images: 30-60 minutes
- Full customization: 1-3 hours total

Most people spread this over a few days, which works great!

### "Can I change it later?"
Absolutely! You can update content, colors, and images anytime.

### "What about mobile phones?"
Everything automatically adjusts to look great on phones, tablets, and computers.

### "Can I have the site in multiple languages?"
Yes! The CUSTOMIZATION-GUIDE shows you how to add French (or any other language).

## Tips for Success

**Take Your Time:**
Work through one guide at a time. Don't rush. It's okay to take breaks!

**Save Your Work:**
Press Ctrl+S (or Cmd+S on Mac) after every change. Save often!

**Preview Frequently:**
Run `npm run dev` and check your changes in the browser. See changes as you make them!

**Get Feedback:**
Show friends, family, or customers and get their input before launching.

**Don't Worry:**
You can't permanently break anything. If something goes wrong, you can always revert to the original template.
